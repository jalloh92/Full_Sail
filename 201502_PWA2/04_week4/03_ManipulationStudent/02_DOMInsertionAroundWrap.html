.wrap( wrappingElement )

~wrappingElement
Type: Selector or htmlString or Element or jQuery A selector, element, HTML string, or jQuery object specifying the structure to wrap around the matched elements.

.wrap( function )
~function
Type: Function( Integer index ) => String or jQuery A callback function returning the HTML content or jQuery object to wrap around the matched elements. Receives the index position of the element in the set as an argument. Within the function, this refers to the current element in the set.

Syntax
$(selector).wrap(wrappingElement,function(index))
