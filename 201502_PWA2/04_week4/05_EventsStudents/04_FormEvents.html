.trigger( eventType [, extraParameters ] )
~eventType
Type: String A string containing a JavaScript event type, such as click or submit.

~extraParameters
Type: Array or PlainObject Additional parameters to pass along to the event handler.

.trigger( event [, extraParameters ] )
~event Type: Event A jQuery.Event object.

Syntax
Trigger the change event for the selected elements:
$(selector).change()
or
$(selector).change(function)

Write something in the input field, and then press enter or click outside the field.
